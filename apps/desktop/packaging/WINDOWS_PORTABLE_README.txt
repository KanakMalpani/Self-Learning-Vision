Self-Learning Vision - Windows Portable Alpha
=============================================

This is an unsigned alpha build. Windows SmartScreen may warn that the
publisher is unknown.

To run:
1. Keep all files in this folder together.
2. Open "Self-Learning Vision.exe".
3. The local engine starts on 127.0.0.1 and stores data in your Windows
   app-data folder, not beside this executable.

The portable app requires Microsoft Edge WebView2 Runtime. Most current
Windows systems already have it. If it is missing, install WebView2 from
Microsoft or use the setup installer, which includes its bootstrapper.

This desktop alpha uses local-only privacy defaults and does not enable
hosted recognition providers automatically.
